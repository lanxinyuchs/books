AutoIntegration

AutoIntegration user interface is used to perform a node integration.
Semi-auto integration is performed in two steps.
First step is to start downloading site installation file by pressing "Download files" button.
Second step is to start the integration by pressing the "Integrate" button.
The download and integration procedure can be monitored in the AutoIntegration log area.
A started download or integration procedure can be cancelled by pressing "Cancel" button.
The AutoIntegration log can be exported by pressing "Export AutoIntegration log" button.
