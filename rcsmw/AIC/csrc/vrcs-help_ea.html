Emergency Access

The emergency access user interface contains Export ESI package and Node Information.

Export ESI package
This function is used to perform a data collection when experiencing problems with the product. The data collection is performed by exporting the Ericsson Support Information (ESI) package. The ESI package is to be enclosed in a Customer Service Request (CSR) and sent to Ericsson for analysis.
Note: Perform data collection as soon as possible, before any recovery is attempted, as the recovery procedure results in some logs and statistics being overwritten.

Node Information
Node status - shows the current status of the node. "Node is working" indicates a node which are working properly.
