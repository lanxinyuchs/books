Files in this directory are:

imm_objects_ref.txt      Content of the imm_objects table after running
                         simple_upgrade.sh. To produce this and similar
                         files, run the dump:dtf() function. The 'dump'
                         module is in the CNX*/test/suites directory.


objects_diff.sh          Compares imm_objects_ref.txt and another supposedly
                         equivalent file

SwmInternal.1            A recent copy of the SwmInternal file from running
                         simple_upgrade.sh.
